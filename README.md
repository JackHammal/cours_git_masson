# Notes cours de Git

## Github

1. Créer un compte Github
2. Créer un repository
3. On prend l'url du repository

## Sur la machine

Créer un dossier pour faire le projet

```
mkdir cours_masson
cd cours_masson
```

On active git

`git init`

On verifie la version de git

`git --version`

On connecte le dossier à notre Github

`git remote add origin https://github.com/JackHammal/cours_git_masson.git`

On crée et pousse un fichier

`git add text.txt`

`git config user.email "git@j123hammal@gmail.com"`

`git config user.name "JackHammal"`
